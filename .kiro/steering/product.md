# Product Overview

This is a comprehensive Flutter ERP (Employee Management System) application that provides:

## Core Features
- **Biometric Authentication**: Fingerprint and face recognition with PIN fallback
- **Employee Management**: Full CRUD operations with search, filtering, and bulk operations
- **Attendance Tracking**: Clock in/out with geolocation and time tracking
- **Payroll Management**: Automated calculations, payslip generation, and history
- **Leave Management**: Leave request system with approval workflows
- **Offline-First Architecture**: Full functionality without internet connectivity

## Key Characteristics
- **Security-First**: Biometric auth, secure token storage, JWT authentication
- **Offline Capable**: SQLite local storage with automatic sync when online
- **Mobile-Optimized**: Responsive Material Design for all device sizes
- **Enterprise-Ready**: Bulk operations, CSV import/export, comprehensive reporting

## Target Users
- HR departments and managers
- Employees for self-service operations
- Administrators for system management

## Business Value
Streamlines employee management processes, reduces manual work, ensures accurate attendance and payroll processing, and provides offline reliability for field workers.
